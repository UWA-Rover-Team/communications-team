Navigation

Tabs: Arm, Drivetrain.
Mode Toggle: Switch active control target (Arm ↔ Drivetrain).
Connection Indicator: Connected/disconnected + latency.
Last Command ID: Incrementing counter for sent commands.
Global Controls

E-Stop (Combined): Stops arm and drivetrain; clear/reset control.
E-Stop (Separate): Arm E-Stop, Drivetrain E-Stop; clear/reset controls.
Status Banner: Shows active/E-stopped/error states for each subsystem.
Arm Tab

Sliders (Cartesian): X, Y, Z; live value readouts.
Slider (Wrist): Wrist orientation angle.
Preset Buttons: Home, Stow, Pickup, Extended; Save Current, Load Custom.
Joint Angles (Read-only): Base, Shoulder, Elbow, Wrist; Homed/Calibrated flags.
2D Arm Visualization: Top-down current position + target; optional deadzone overlay.


Drivetrain Tab
Sliders: Throttle (forward/back), Turn (left/right).
Speed Level Selector: Slow/Normal/Fast.
Per-Motor Status: FL(251), FR(253), BL(254), BR(252) RPM/current/temp.
Odometry: X, Y, Heading, Speed.
Battery/Bus Voltage: Live readout.

Joystick & Input
Joystick Mode Indicator: Shows current target of the Logitech Extreme 3D Pro.
Deadzone Control: Slider 0.05–0.30; enable/disable visualization.
Keyboard Shortcuts: Mode toggle (Tab), presets (1–4), E-stops (Space/Shift+A/Shift+D).
Telemetry & Diagnostics

Arm Telemetry Panel: Joint angles, end-effector XYZ + wrist, currents, temps, fault codes.
Drivetrain Telemetry Panel: Motor RPMs/currents/temps, total current, odometry, battery.
Update Rate Display: Telemetry rate and last update time.
Logging & Console

Log Viewer Button: Opens live logs; filter by subsystem.
Console Output: Rolling messages for connects, ACKs, errors, state changes.
Settings

Deadzone Slider: Axis deadzone value; checkbox to show overlay.
Arm Limits: Max joint speed; toggles for angles/diagnostics visibility.
Drivetrain Limits: Max speed %, acceleration %; odometry toggle.
Network: Rover IP/port; auto-reconnect toggle.
Logging: Enable/disable; log directory; show console toggle.
